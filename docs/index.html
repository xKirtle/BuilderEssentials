Auto Hammer example usage

Auto Hammer
Mouse ButtonDescription
LeftHammer tiles
RightOpen slope selection menu

Allows the user to select a custom slope instead of cycling through them all. It'll behave like a vanilla hammer if no custom slope is selected.

Fill Wand example usage

Fill Wand
Mouse ButtonDescription
MiddleClick on a tile to select it as the working type of tile
LeftTile placement in selected region
RightUndo last placement

Great to fill huge holes.
The selection region range must be increased/decreased with the use of a hotkey that is DISABLED by default in tModLoader.

MaterialsStation
1x + 1x

Infinite Paint Bucket example usage

Infinite Paint Bucket

If present in the inventory, will provide infinite paint to the user.
To be used with vanilla painting tools, you still need to make the paint you want to use the first one in your inventory.
